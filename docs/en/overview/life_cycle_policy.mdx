# Plugin Life Cycle Policy

## Release Policy

A new version of the `Alauda Build of SonarQube` plugin is released every 1 year, updating the plugin to the newest [LTA](https://www.sonarsource.com/products/sonarqube/downloads/lts/) version officially released by SonarQube.

::: tip Difference Between Alauda and SonarQube Versions
The SonarQube plugin versions released by Alauda are typically 1 to 12 months behind the latest official SonarQube release.
:::

## Maintenance Policy

Alauda provides 18 months of maintenance support for each released version of the SonarQube plugin.

During the maintenance period, Alauda will provide the following services to customers:

1. Track the patch versions released by SonarQube and promptly provide patches containing bug fixes and security updates.
2. Provide security updates that comply with Alauda's security standards.
3. Assist customers in smoothly upgrading the SonarQube plugin to new versions.

Additionally, Alauda will release a security update every two months during the maintenance period to ensure the security and stability of the plugin.
