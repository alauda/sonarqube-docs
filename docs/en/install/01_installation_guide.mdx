# Installing Operator

## Downloading

:::info

`Alauda Build of SonarQube` Operator can be retrieved from Customer Portal.

Please contact Consumer Support for more information.

:::

## Uploading the Operator

For more information on uploading the operator, please refer to <ExternalSiteLink name="acp" href="ui/cli_tools/index.html#uploading-operators" children="Uploading Operators" />

## Installing Operator

After the Operator is uploaded, you can go to the `Administrator` -> `Marketplace` -> `Operator Hub` page, switch to the target cluster, and then deploy the `Alauda Build of SonarQube` Operator.
